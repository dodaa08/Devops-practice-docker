# Docker Practice Repository

This repository contains examples and exercises created while learning Docker. The goal is to understand how containerization works and how to use Docker for building, running, and deploying applications.

## Contents

1. Basic Dockerfile examples  
2. Building and running images  
3. Working with containers  
4. Using Docker Compose  
5. Simple CI/CD workflow examples  
6. Full stack, Node.js CI/CD, Frontend, Turbo mono repo practice

## What I Learned

1. Writing Dockerfiles for simple applications  
2. Building images using the docker build command  
3. Running containers and exposing ports  
4. Using docker ps, docker images, and docker logs  
5. Understanding volume mounting for local development  
6. Creating multi-service setups using Docker Compose  
7. Pushing images to Docker Hub  
8. Setting up GitHub Actions to build and push Docker images automatically


## How to Use

1. Clone the repository  
2. Navigate to any folder  
3. Read the README inside that folder  
4. Build and run the Docker example

## Goals

1. Continue adding more Docker exercises  
2. Add Kubernetes practice in the future  
3. Document each experiment with clear explanations
